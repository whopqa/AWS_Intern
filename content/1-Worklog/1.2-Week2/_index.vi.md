### Mục tiêu tuần 2:
* Hiểu sâu hơn về kiến trúc VPC và các thành phần: Subnet, IGW, NAT Gateway, Route Table, Security Group.  
* Nắm được cơ chế giám sát và phân tích mạng qua VPC Flow Logs & Reachability Analyzer.  
* Triển khai EC2 và thực hành truy cập bằng Session Manager.  
* Làm quen và thực hành mô hình VPC Site-to-Site.  
* Nắm khái niệm Infrastructure as Code và ứng dụng vào hạ tầng AWS.  
* Cấu hình Hybrid DNS với Route 53 Resolver.  
* Tìm hiểu RDGW – Remote Desktop Gateway trong môi trường Hybrid.  
* Thực hành Transit Gateway và routing đa VPC.  
* Học lý thuyết Module 3: EC2, EBS, AMI, Auto Scaling, metadata, pricing option, EFS, FSx, Lightsail, MGN.  
* Hoàn thành các lab về AWS Backup, Resource Tagging, CloudWatch Monitoring.  
* Triển khai ứng dụng FCJ Management trên Auto Scaling Group.  
* Thực hành Lightsail để tối ưu chi phí.

### Kết quả đạt được tuần 2:

* **Hoàn thành tất cả các lab về VPC và Networking nâng cao**, bao gồm:  
  - Tạo VPC đầy đủ với IGW, NAT Gateway, Route Table tùy chỉnh  
  - Thiết lập Security Group đúng chuẩn  
  - Kích hoạt VPC Flow Logs  
  - Kiểm tra kết nối với Reachability Analyzer  

* **Triển khai EC2** và thực hành các kỹ thuật truy cập hiện đại:  
  - SSM Session Manager (không cần SSH key)  
  - CloudWatch Monitoring cho CPU, Network, Log Insights  

* **Hoàn thành mô hình VPC Site-to-Site** và hiểu rõ cách routing giữa các mạng.

* **Học và thực hành Infrastructure as Code**, kết hợp Route 53 Resolver xây dựng Hybrid DNS.

* **Nắm RDGW** và hiểu cách truy cập tài nguyên trong mô hình Hybrid.

* **Hoàn thành Transit Gateway** và routing đa VPC.

* **Học toàn bộ Module 3**:  
  - AMI, EBS, EC2, Auto Scaling  
  - Instance Store, Metadata/User Data  
  - Pricing Option (On-demand, Reserved, Spot)  
  - Lightsail, EFS, FSx, MGN  

* **Hoàn thành Lab 13 – AWS Backup**, xây dựng giải pháp bảo vệ dữ liệu hoàn chỉnh.

* **Sử dụng Tag và Resource Group** để quản lý tài nguyên hiệu quả.

* **Hoàn thành CloudWatch Workshop**, tạo Dashboard, Alarm, Metric Filter.

* **Triển khai ứng dụng FCJ Management bằng Auto Scaling Group**, đảm bảo HA và scaling tự động.

* **Hoàn thành Lightsail Workshop – Cost Optimization**, hiểu cách tối ưu chi phí cho workload nhỏ.
